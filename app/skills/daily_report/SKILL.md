# Daily Report Skill

## 用途

根据今日统计、近期事件和 memory summary 生成办公健康日报。

## 触发条件

- 用户请求日报、总结、今日回顾或明日重点。
- 定时任务需要生成每日办公健康总结。
- Agent 需要把已有统计和事件整理为结构化报告。

## 输入

- `today_summary`: 今日统计，必须来自统计工具。
- `recent_events`: 近期事件，必须来自事件日志。
- `memory_summary`: 用户历史摘要，可为空。

## 输出

- `report_title`: 日报标题。
- `summary`: 总结。
- `highlights`: 关键发现。
- `suggestions`: 建议。
- `tomorrow_focus`: 明日关注点。

## 禁止事项

- 不编造未记录的统计数据。
- 不生成医疗诊断。
- 不夸大风险。
- 建议必须能对应到已有事件、统计或 memory。

## 示例

输入：今日有久坐提醒和饮水提醒。

输出：总结今日提醒次数，并建议明天提前安排短暂起身活动、把水杯放在更显眼的位置。
